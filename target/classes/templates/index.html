About This Site

This is a simple web application built with Java 21, Spring Boot, Maven, and Thymeleaf. It's styled using Bootstrap 5 and deployed on a server running Ubuntu.

Spring Boot

Easily build stand-alone, production-grade Java web services.

Responsive design and ready-to-use UI components for a modern experience.
